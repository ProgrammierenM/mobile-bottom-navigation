# Mobile Bottom Navigation

Mobile Navigation mit HTML CSS und JavaScript. Mit diesem kleinen Projekt programmieren wir eine Mobile Bottom Navigation welche bequem mit einer Hand auf dem Smartphone bedienbar ist. Das aktive Element der Navigation passt sich per Animation automatisch an. Einmal per Klick auf einen Link und einmal anhand der Scroll Position auf der Seite.
